# 배열(Array)

```
var WepProglam = ['To_ToRo','godofkoding','bw-99'];
```
>console.log(WepProglam[0]);
>> 실행결과: To_ToRo<br/>

>console.log(WepProglam.length); #크기
>> 실행결과: 3

배열의 끝에 원소 하나 추가
```
WepProglam.push('avactor');
```

배열의 끝에 복수의 원소 추가
```
WepProglam = WepProglam.concat(['GirlHumanFriend'sBoyFriendPuncher','Lee U Kyoung']);
```

배열의 시작점에 원소 하나 추가
```
WepProglam.unshift('zzz');
```

첫번째 원소 제거
```
WepProglam.shift();
```

마지막 원소 제거
```
WepProglam.pop();
```

순서 정렬
```
var li = ['b','e','a','d','c'];
li.sort();
console.log(li);
```
>실행결과:
>>>(5) ["a","b","c","d","e"]

역순 정렬
```
var li = ['b','e','a','d','c'];
li.reverse();
console.log(li);
```
>실행결과:
>>(5) ["c","d","a","e","b"]
